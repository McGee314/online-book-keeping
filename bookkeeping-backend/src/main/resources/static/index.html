Backend Running

Personal Online Bookkeeping System

The Spring Boot backend is up. This landing page replaces the default Whitelabel Error Page so you can quickly verify the server is working when you open http://localhost:8080/.

Current Day 2 Scope

  • User registration
  • User login
  • JWT authentication
  • Current user info endpoint

Useful Endpoints

  • POST /api/auth/register
  • POST /api/auth/login
  • GET /api/auth/me

Run Commands

cd bookkeeping-backend
./mvnw spring-boot:run

Setup Reminder

Make sure MySQL is running and your environment variables are set before starting the app. 

export DB_PASSWORD='your_mysql_password'
export JWT_SECRET='YourSuperSecretKey...'
